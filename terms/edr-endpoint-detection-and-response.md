# Endpoint Detection and Response (EDR)

## Definition

Endpoint Detection and Response (EDR) is a security solution that continuously monitors endpoints, detects suspicious activity, and provides tools for investigation and response.

## Why It Matters

EDR provides analysts with detailed visibility into endpoint activity and enables rapid containment of threats.

## Example

Microsoft Defender for Endpoint detects suspicious PowerShell execution and automatically isolates the affected device.
