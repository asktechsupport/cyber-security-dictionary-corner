# Security Information and Event Management (SIEM)

## Definition

A Security Information and Event Management (SIEM) platform collects, normalises, stores, and analyses logs from multiple systems to identify potential security threats.

## Why It Matters

The SIEM acts as the central source of visibility within a SOC, allowing analysts to investigate alerts using data from across the organisation.

## Example

Microsoft Sentinel correlates firewall logs, Microsoft Entra ID sign-in logs, and Microsoft Defender alerts to identify a possible account compromise.
