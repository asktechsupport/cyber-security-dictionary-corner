# False Positive

## Definition

A false positive occurs when legitimate activity is incorrectly identified as malicious by a security tool.

## Why It Matters

False positives consume analyst time and contribute to alert fatigue, making it more difficult to identify genuine threats.

## Example

An employee signs into Microsoft 365 while travelling overseas, triggering an Impossible Travel alert that is later confirmed to be legitimate.
