# MITRE ATT&CK

## Definition

MITRE ATT&CK is a globally recognised knowledge base that documents adversary tactics, techniques, and procedures (TTPs) based on real-world observations.

## Why It Matters

Security teams use MITRE ATT&CK to understand attacker behaviour, improve detections, and map incidents to known techniques.

## Example

PowerShell execution is mapped to Technique **T1059 – Command and Scripting Interpreter**.
