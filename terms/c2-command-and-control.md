# Command and Control (C2)

## Definition

Command and Control (C2) is the infrastructure attackers use to communicate with compromised devices. Malware commonly contacts a C2 server to receive instructions or exfiltrate stolen data.

## Why It Matters

Detecting communication with known C2 servers is one of the most common methods of identifying compromised endpoints.

## Example

A workstation infected with RedLine Stealer attempts to connect to a malicious IP address associated with known Command and Control infrastructure.
